# Media Tools Reference

## AI视频提示词公式
**基础**：主体 + 场景 + 风格
**进阶**：主体 + 动作 + 场景 + 环境动态 + 风格 + 画质

### 即梦AI关键技巧
- 运镜：平移/旋转/推拉/希区柯克变焦
- 图生视频：上传参考图→锁定角色→跨集保持
- 尾帧锁定：勾选"使用尾帧"→增强稳定性

### Seedance 2.0管线
```
脚本(Kimi/Claude) → 即梦AI(角色图+@语法) → GPT Image2(故事板) → Seedance 2.0(动态化) → 剪映(后期)
```

### 三镜头骨架模板
- 镜头1(4s,建置)：遮挡滑出+轻俯拍+广角+横移→停住
- 镜头2(6s,升级)：dolly in+中长焦压缩+双人同框
- 镜头3(5s,钩子)：主观POV+推近关键物→停1秒+短台词≤8字

### 提示词三明治(≤250字)
```
第一层：场景一句话（地点+时间+氛围）
第二层：三镜头口令（运镜动词+停顿+音效）
第三层：钩子一句话（关键物+短台词）
```
