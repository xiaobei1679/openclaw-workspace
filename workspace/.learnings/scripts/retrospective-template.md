# 5章复盘会模板
# 当异兽学院章节达到5的倍数时自动触发
# 输出：桌面\QClaw产出文件\_协作基础设施_\复盘_第N章_YYYYMMDD.md

## 触发条件
- 检查 桌面\QClaw产出文件\小说正文\ 下最新章节号
- 章节号 % 5 == 0 → 触发复盘
- 已复盘过的章节号 → 跳过

## 复盘内容

### 1. 字数趋势
统计第N-4到第N章的CJK字数，画趋势（升/平/降）
- 阈值：每章≥2200字，连续2章<2200 → ⚠️

### 2. AI腔趋势
读取第N-4到第N章的诊断报告
- 统计每章AI腔超标段落数
- 改写率趋势
- 连续2章改写率<30% → ⚠️

### 3. 爽点密度分析
读取世界观v2爽点密度要求表
- 第N-4到第N章实际爽点分布
- 是否递进？是否有连续2章无爽点？

### 4. 重复模式检测
读取creation-feedback的patterns输出
- 跨章节重复句式/情节模式
- 已升级为创作规则的：列出
- 新检测到的重复：标注

### 5. 角色发展检查
- 主角林远：性格是否一致？成长弧线是否递进？
- 配角：是否有工具人嫌疑？是否有独立动机？
- 毛球：借力机制是否有递进？代价是否在浮现？

### 6. 经验池消费统计
- 经验池目前有多少条经验？
- 哪些被创作引用过？哪些从未引用？
- 有没有新经验需要record？

### 7. 下5章策略建议
基于以上6项分析，给出：
- A：继续保持的
- B：需要调整的（附具体调整方向）
- C：需要新增的（如新设定补丁、新角色等）
